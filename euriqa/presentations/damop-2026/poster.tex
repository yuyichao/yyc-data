        \begin{center}
          \begin{columns}
            \column{30cm}
            \begin{itemize}
            \item Limitation of quantum simulation type interaction,
              \begin{itemize}
              \item Weak and short interaction range.\\
                Smaller detuning from motional modes
                is needed for stronger and longer-ranged interaction.
              \item Fast modulation of amplitude can lead to closure error
              \end{itemize}
            \item Overcoming these limitations requires
              explicit management of mode closure.
            \item Equivalent to multi-qubit parallel M{\o}lmer-S{\o}rensen gate.
            \item For optimization efficiency, the solution is constructed from
              a linear combinations of precomputed solutions,
              each already guarantee robust motional mode closure.
            \end{itemize}
